# VISOCA — Gestión de Calidad

Sistema de gestión de calidad B2B para LIGS Group.

## Configuración local

Copia `.env.example` a `.env` y completa los valores antes de `bun dev`.

```bash
cp .env.example .env
```

Luego edita `.env` con las credenciales reales de tu proyecto Supabase.

## Tecnologías

- React + TypeScript
- TanStack Start
- Tailwind CSS
- Supabase
